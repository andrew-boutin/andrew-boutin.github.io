I ♥ Video Games


“A game is a series of interesting choices.” – Sid Meier



I'm addicted to playing and creating video games! Here are some of the things that I've worked on.


Gravity Clutch Racing

A multiplayer vehicle racing game where you can use the gravity clutch to alter the direction of gravity while racing!

Geo Wars

Play Geo Wars on Kongregate. Requires either IE11 or Safari with WebPlayer (Unity stopped supporting WebPlayer so the game most likely won't load on the site). If you have a Kongregate account then you can get your high score saved on the site. This is a simple 2D platformer shooter. You have to avoid fatal terrain and different kinds of enemies to beat the levels. This was a group project where I did all of the game dev except for the concept and art work. I can't share the source code because of this.

Crazy Canal Tower Defense

A tower defense game where your objective is to strategically place defense towers to prevent enemy water craft from reaching the end of the canal.

Squares Clone

This is a clone of the popular Squares 2 game. Black and red blocks move across the screen at varying speeds. The object of the game is for the player to collect black squares. They gain points and their size increases. If the player hits a red square the game ends. The speed of all the blocks increases as the game progresses. It's a very simple but addicting game.

Epic Crawl

This is a 2D top down RPG that features a level designer for players to build their own adventure. Currently only the level designer is playable.

Idle Fidget Spinners

Play Idle Fidget Spinners! Create different kinds of fidget spinners and keep them spinning to earn money. Earn money to buy more fidgt spinners of course. This is a classic idle progression game.

Labyrinth Crawl

Download LabyrinthCrawl to play! This is a terminal based strategy game where you try to escape to labyrinth while fighting enemies along the way. The mazes are randomly generated so each time you play is different. Let me know if you beat all 10 levels!

Squares Clone

python
pygame

This is a clone of the popular Squares 2 game. Black and red blocks move across the screen at varying speeds. The player's cursor is also a black square. The object of the game is for the player to collect other black squares with their cursor. They gain points and their cursor size increases. If the player hits a red square the game ends. The speed of all the blocks increases as the game progresses. It's a very simple but addicting game.

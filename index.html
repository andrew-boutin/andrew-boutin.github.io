Slack Bot

python, slack

The EMMM (eeny, meeny, miny, moe) Slack Bot. Set this Slack bot up with your Slack workspace and you'll be able to "@emmm". Emmm will choose a random user from the channel and respond with their @tag. There are some configuration options to do things like ignore other bots present in the channel so they aren't chosen.

Squares Clone

python, pygame

This is a clone of the popular Squares 2 game. Black and red blocks move across the screen at varying speeds. The player's cursor is also a black square. The object of the game is for the player to collect other black squares with their cursor. They gain points and their cursor size increases. If the player hits a red square the game ends. The speed of all the blocks increases as the game progresses. It's a very simple but addicting game.

Geo Wars

unity, c#, kongregate-api

Play Geo Wars! The game is hosted on Kongregate and requires FireFox with WebPlayer. If you have a Kongregate account then you can get your high score saved on the site. This is a simple 2D platformer shooter. You have to avoid fatal terrain and different kinds of enemies to beat the levels. This was a group project where I did all of the game dev except for the concept and art work. I can't share the source code because of this.
